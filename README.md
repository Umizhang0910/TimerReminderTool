# TimerReminderTool
This is a timer reminder tool. Its purpose is to allow you to take short breaks during your busy coding work, achieving a balance between work and rest.</br></br>
这是一个定时提醒工具，目的是让你在繁忙的编码工作中间歇的休息下，劳逸结合~
